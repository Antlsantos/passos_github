Como Criar uma Conta no GitHub e Criar Seu Primeiro Repositório

Passo 1: Criar uma Conta no GitHub

  1. Acesse o site oficial do GitHub.
  2. Clique em "Sign up" no canto superior direito.
  3. Preencha o nome de usuário, e-mail e senha.
  4. Complete a verificação de segurança.
  5. Escolha o plano (você pode começar com o plano gratuito).
  6. Clique em "Create Account" para finalizar.

A imagem acima é o logotipo do GitHub, o site que você usará para criar e gerenciar seus repositórios.

Passo 2: Criar Seu Primeiro Repositório

  1. Após logar no GitHub, clique no ícone de "+" no canto superior direito e selecione "New repository".
  2. Dê um nome para o seu repositório (exemplo: "meu-primeiro-repositorio").
  3. Selecione a opção "Public" para tornar o repositório visível a todos.
  4. Marque a opção "Initialize this repository with a README".
  5. Clique em "Create repository" para criar o repositório.

Este é o botão onde você clica para criar um novo repositório após preencher as informações solicitadas.

Passo 3: Obter o Link do Repositório

Após a criação do repositório, o link do seu repositório estará disponível na barra de endereços do navegador. O formato será:

https://github.com/seu-usuario/meu-primeiro-repositorio

Substitua "seu-usuario" pelo seu nome de usuário no GitHub e "meu-primeiro-repositorio" pelo nome do seu repositório.

Na imagem acima, você verá a estrutura de um repositório no GitHub. O link para seu repositório estará visível na barra de endereços do navegador.
